## AI Trade v0.3.0

本版本把 v0.2.0 的自动交易执行能力继续升级为智能量化交易总控台，覆盖交易所情报、社区账号、链上异动、现货/合约价差、策略命中和大模型综合分析。

### Highlights

- 新增智能总控台页面：`/terminal`
- 新增总控台模块页面：`/terminal/market`、`/terminal/community`、`/terminal/onchain`、`/terminal/basis`、`/terminal/strategies`、`/terminal/trading`、`/terminal/risk`
- 新增总控台快照 API：`/api/terminal/snapshot`
- 新增总控台模块 API：`/api/terminal/modules/{module}`
- 新增平台能力 API：`/api/platform/capabilities`
- 新增平台账户、策略、风控和日志 API：
  - `/api/platform/accounts`
  - `/api/platform/strategies`
  - `/api/platform/risk`
  - `/api/platform/logs`
- 新增 `IntelligenceHub` 数据层，统一聚合多源监控数据
- 新增 `PlatformSnapshot` 平台层，展示接入层、策略层、执行层、数据层、风控层状态
- 支持交易所关键情报与热门信息筛选
- 支持 Binance / OKX 接入状态监控，OKX 密钥可在 `/settings` 配置
- 支持 Twitter/X tracked accounts 配置画像与状态监控
- 支持链上大额异动监控：CSV 接入与实时信号代理 fallback
- 支持现货 / 合约价差分析：CSV 接入与行情推导 fallback
- 支持策略命中聚合：自动交易阈值、量能、买盘压力和候选动作
- 支持账户概览、策略目录、风控规则和交易日志在总控台集中展示
- 支持在 `/terminal/trading` 直接运行一次模拟量化交易，强制 paper 模式，不提交真实订单
- 新增模拟交易 API：`POST /api/trading/paper/run`
- 新增系统级中英文切换：`?lang=zh` / `?lang=en`
- 关键指标、菜单、表头、状态和自动交易事件支持中文通俗展示与英文专业术语展示
- 全局 UI 升级为交易终端布局：左侧模块菜单、顶部功能导航、暗色数据面板和底部行情条
- 支持 OpenAI Responses API 综合分析
- OpenAI 未配置或调用失败时自动回退本地规则分析
- 设置页新增 OKX Auth 与 Intelligence & LLM 配置项
- UI 升级为更接近交易所量化终端的总控台布局

### Data Sources

- `data/exchange_intel.csv`
- `data/onchain_events.csv`
- `data/futures_basis.csv`

示例文件：

- `data/exchange_intel.example.csv`
- `data/onchain_events.example.csv`
- `data/futures_basis.example.csv`

### LLM Integration

使用 OpenAI Responses API：

- Endpoint: `POST /v1/responses`
- Config: `OPENAI_API_KEY` / `OPENAI_MODEL`
- Runtime settings: `/settings` -> `Intelligence & LLM`

### Validation

- `PYTHONPATH=src python3 -m unittest discover -s tests`
- `PYTHONPATH=src python3 -m py_compile ...`
- `/terminal` HTTP 200
- `/terminal/{module}` HTTP 200
- `/api/terminal/snapshot` HTTP 200
- `/api/terminal/modules/{module}` HTTP 200
- `/api/platform/capabilities` HTTP 200
- `POST /api/trading/paper/run` 完成 paper 模拟交易
- `/terminal/trading?lang=zh` 显示中文状态
- `/terminal/trading?lang=en` 显示英文状态
- `POST /terminal/trading/run?lang=en` 显示英文执行结果
- 扫描、回测、自动交易、设置和总控台页面均返回新版 `app-shell` 终端外壳
