## AI Trade v0.1.0

首个可公开使用版本，重点把“扫描、回测、舆情、配置”收敛到同一个本地工作台里。

### Highlights

- 实时扫描 Binance Spot 高流动性交易对
- 技术指标打分：RSI / EMA / MACD / KDJ / 量能 / 主动买盘
- 历史回测：单币种、组合横截面、手续费、滑点、资金曲线、最大回撤
- 社区情报源：X/Twitter、CSV、新闻 CSV、Telegram CSV、Reddit
- 回测结果导出：JSON / CSV
- 内建回测模板：`balanced_swing`、`breakout_aggressive`、`portfolio_rotation`
- 运行配置页：统一配置密钥、情报源和默认策略参数
- 配置模板导入 / 导出
- 可选加密配置存储：支持 `RUNTIME_CONFIG_PASSPHRASE`

### Web Entrypoints

- `/` 实时扫描
- `/backtest` 历史回测
- `/settings` 运行配置
- `/api/scan` 扫描 API
- `/api/backtest` 回测 API
- `/api/backtest/export` 回测导出
- `/api/backtest/presets` 回测模板清单

### Notes

- 默认兼容本地明文 `data/runtime_config.json`
- 设置 `RUNTIME_CONFIG_PASSPHRASE` 后，后续保存会自动写成加密格式
- 本项目不负责自动下单，主要用于研究、筛选和验证
